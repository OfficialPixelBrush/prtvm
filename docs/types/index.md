# Types

The following is a list of how all types that exist in the PRTVM.

| Label | Full name | Function |
| --- | --- | --- |
| `bool` | Boolean | Used for binary `true` or `false` variables |
| `i8` | Byte | Used for signed 8-Bit numbers |
| `i16` | Short | Used for signed 16-Bit numbers |
| `i32` | Integer | Used for signed 32-Bit numbers |
| `i64` | Long | Used for signed 64-Bit numbers |
| `f32` | Float | Used for signed 32-Bit floating-point numbers |
| `f64` | Double | Used for signed 64-Bit floating-point numbers |
